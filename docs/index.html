OpenBiliClaw vNext

让内容发现围绕证据，而不是平台黑箱。

OpenBiliClaw 把 Bilibili、小红书、抖音、YouTube、X、知乎和 Reddit 的受支持只读能力 归一为活动证据与候选内容，再生成可追溯画像和 Feed。Web 与浏览器扩展共享同一套 /api/v1 合同；收藏与稍后观看只保存在本地。

Evidence profile

导入、反馈、聊天学习与用户编辑统一为 revisioned profile evidence。

Typed execution

PydanticAI 负责结构化输出，LiteLLM 负责 provider、路由、重试、预算与缓存。

Durable jobs

Huey 与应用 SQLite 分离；用户可见任务状态由应用数据库持久化。
