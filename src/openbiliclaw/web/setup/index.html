连接浏览器扩展和平台账号

让 OpenBiliClaw 读到你在各平台的真实行为,推荐才懂你。最省事的方式是装浏览器扩展,它会在初始化时使用当前浏览器里已登录的平台账号。

  1. 安装 OpenBiliClaw 浏览器扩展（Chrome / Edge）
  2. 在浏览器里登录要接入的平台账号，例如 bilibili.com / reddit.com
  3. 下一步只会检查你勾选的平台，B 站默认推荐但可以取消
